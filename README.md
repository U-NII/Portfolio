# OFFING

## サイト概要

サイト内で主催者、日程、人数、店、を決めてオフ会をセッティングするサービス。

### サイトテーマ

オフ会をセッティングする

### テーマを選んだ理由

①類似した専用サービスがなかったため、こういうサービスがあるとどういう反応があるか知りたい。
②共通の趣味を持った人との新しいコミュニケーションツールとして開発してみたかった。

### ターゲットユーザ

20~30 代

### 主な利用シーン

共通の趣味、話題でオフ会を通じて盛り上がりたい人に向けて
